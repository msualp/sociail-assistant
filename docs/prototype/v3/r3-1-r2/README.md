# v3 R3.1-r2 Holistic Exterior Redesign (MagSafe)

Generated print assets for the v3.1-r2 holistic redesign: stronger all-around corner rounding and full-body trapezoidal draft shaping.

## What Changed vs R2
- Larger perimeter edge rounds and softened crown transitions
- Added subtle side/shoulder contouring for improved in-hand comfort
- Added MagSafe pocket alignment bossing + anti-rotation key
- Upgraded seam to 6 snap points for stronger, more even retention
- Tuned clearances for repeatable assembly/disassembly on FDM prints
- Geometry tuned to print cleanly in recommended orientations with minimal supports

- `shell-top.stl`, `shell-bottom.stl`
- `shell-top-print.stl`, `shell-bottom-print.stl`
- `shell-top-print-screws.stl`, `shell-bottom-print-screws.stl`
- `neg-magsafe-ring.stl`, `neg-led-window.stl`, `neg-button.stl`
- `neg-usbc-bottom.stl`, `neg-mic-array.stl`, `neg-speaker-grill.stl`
- `neg-screw-holes-4x.stl`

## Print Orientation
- Top: seam-down (open cavity up, split plane on bed)
- Bottom: upright (flat back on bed)

## Bambu Slicer Baseline
- Layer height: `0.16` or `0.20`
- Walls: `4`
- Top/Bottom layers: `6`
- Supports: `off` (enable only if your machine/profile bridges poorly near side I/O)
